# AI Resume Screening Tool

## Overview

AI Resume Screening Tool is a web application built using Python and Flask that helps users analyze their resumes against a job description. The system calculates an ATS (Applicant Tracking System) score, identifies matched skills, highlights missing skills, and provides resume insights through a modern dashboard interface.

## Features

* ATS Score Calculation
* Resume Upload (PDF/DOCX)
* Resume Preview
* Matched Skills Detection
* Missing Skills Analysis
* Resume History Storage
* SQLite Database Integration
* Responsive Dark Theme UI
* Animated ATS Score Dashboard

## How It Works

1. Upload a resume in PDF or DOCX format.
2. Paste the job description.
3. Click "Analyze Resume".
4. The system calculates an ATS score.
5. View matched skills and missing skills.
6. Resume data is stored in the database for future reference.
